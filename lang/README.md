# الترجمات (عربي / إنجليزي)

## الملفات

- **`en.json`** — الإنجليزية (والمفتاح يُعاد كقيمة افتراضية إن لم تُوجد ترجمة)
- **`ar.json`** — العربية

اللغة الحالية تُحدد من `config/app.php` أو من middleware اللغة (`APP_LOCALE` أو من الرابط `/lang/{locale}`).

## عند إضافة شيء جديد

1. استخدم دوماً **`__('Key')`** للنصوص في الـ views والـ controllers (رسائل نجاح/خطأ).
2. أضف المفتاح **بنفس اللفظ الإنجليزي** في **`lang/en.json`** (القيمة بالإنجليزي).
3. أضف **نفس المفتاح** في **`lang/ar.json`** (القيمة بالعربي).

مثال:

```php
// في الكود
__('Add User')
```

في **en.json**:
```json
"Add User": "Add User"
```

في **ar.json**:
```json
"Add User": "إضافة مستخدم"
```

بهذا يصبح كل واجهة جديدة مترجمة تلقائياً حسب لغة التطبيق.
